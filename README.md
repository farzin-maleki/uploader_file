# Simple File Uploader

A simple one-page file uploader interface built to practice front-end layout and styling using HTML and CSS.

## Technologies Used
- HTML5
- CSS3

## Features
- Clean one-page layout
- Simple file upload interface
- Basic user-friendly design

## Purpose of the Project
This project was created as part of my front-end development practice to improve my understanding of UI layout and basic web structure.
